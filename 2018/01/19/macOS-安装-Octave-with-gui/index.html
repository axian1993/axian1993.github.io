#可行的方法
使用 MacPorts 安装，安装步骤：

  1. 安装MacPorts（若安装完后无法使用MacPorts，尝试手动修改环境变量)

  2. 更新MacPorts

    sudo port selfupdate

  3. 安装Octave

    sudo port octave

安装完成后在命令行输入Octave启动
